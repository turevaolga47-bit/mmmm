# Логирование проекта "ТЭС Ассистент"

## Описание проекта
Персональный ассистент по Технике Эмоциональной Свободы (ТЭС/EFT) от Ольги Турьевой.

## Целевая аудитория
Взрослые женщины 35–80 лет

## Основные функции

### 1. Главная страница (Dashboard)
- Приветствие пользователя
- Быстрый доступ к разделам: Консультация, Упражнения, Архив, Эксперт
- Список тем для проработки (первые 4 темы)

### 2. Консультация (Chat/Assistant)
- Чат с AI-ассистентом
- Ввод сообщений пользователем
- Ответы от ассистента (заглушка, требуется API ключ)

### 3. Упражнения (Exercises)
- Список техник ТЭС с тэппинг-скриптами
- Установочные фразы
- Раунды тэппинга для каждой темы

### 4. Архив (Education)
- Полный список тем для проработки
- Детальная информация по каждой теме
- Тест на уровень стресса

### 5. Профиль эксперта (Expert)
- Информация об Ольге Турьевой
- Ссылки на Telegram и канал
- Описание закрытого женского клуба
- Информация о технике ТЭС

### 6. Тест на стресс (StressTest)
- 10 вопросов о стрессе
- 5 вариантов ответа на каждый вопрос
- Расчёт уровня стресса от 1 до 10
- Цветовая индикация результата
- Рекомендации по результатам
- Ссылки на консультацию и канал

### 7. Дневник рефлексии (Journal)
- Заглушка "будет доступен после 40-дневного протокола"

## Темы для проработки

1. **Тревога** - Освобождение от деструктивного ожидания угроз
2. **Выгорание** - Регенерация жизненных сил и смыслов
3. **Стресс** - Мгновенная нейтрализация
4. **Усталость** - Активация меридианов
5. **Самопознание** - Манифестация успеха
6. **Мониторинг стресса** - Тест из 10 вопросов

## Структура файлов

```
/
├── App.tsx                    # Главный компонент с роутингом
├── index.tsx                  # Точка входа React
├── index.html                 # HTML шаблон
├── types.ts                   # TypeScript типы
├── constants.tsx              # Константы (темы, информация эксперта)
├── vite.config.ts             # Конфигурация Vite

    ├── Layout.tsx             # Обёртка с навигацией

    ├── Assistant.tsx          # Чат с ассистентом
```
